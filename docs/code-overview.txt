Canvas App
Создание приложений на JavaScript

Canvas App — это тип смартапа с привлекательным интерфейсом и виртуальным ассистентом. Управлять Canvas App можно разными способами: голосом, текстовыми сообщениями, жестами, касанием и пультом. Использовать Canvas App можно на следующих поверхностях:

    Салют ТВ;
    SberBox;
    SberBox Time;
    мобильное приложение Салют.

Вы можете увидеть реальные примеры Canvas App на своих устройствах. Например, попробуйте запустить пошаговые рецепты Супершеф или закажите продукты через SberBox.

Code — это среда разработки на языках JavaScript и SmartApp DSL. С помощью Code вы получаете доступ к редактору интентов, сущностей и сценариев, а также к развитым аналитическим инструментам.

Чтобы использовать Code, зарегистрируйтесь в Studio﻿

и создайте проект Code.
Интерфейс Code

Проекты, разрабатываемые в Code, отмечаются в списке приложений ярлыком Code.
Технические требования﻿

Смартапы, созданные в Code, будут работать на всех устройствах, независимо от браузера и типа устройства.

Однако для создания смартапа необходимо соблюсти ряд требований.
Google Chrome﻿
Code работает только в браузере Google Chrome﻿

. На данный момент мы не тестировали функциональность в других браузерах и не можем гарантировать корректную работу. Поэтому используйте Google Chrome.
Использование на ПК﻿

Для работы со средой разработки Code необходимо использовать ноутбук или стационарный компьютер.
Стабильный интернет﻿

Для корректной работы вам потребуется стабильное подключение к сети Интернет.

У браузера Google Chrome, в свою очередь, есть собственные требования. Ознакомиться с ними можно на этой странице﻿
